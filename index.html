Decant a build straight into your game

Decant drops a ready-made build into Path of Exile 2's in-game Build Planner — no copy-paste, no alt-tabbing, no spreadsheets. Connect your folder once and every build on this page is a single click away.

  1. 1

    Connect your folderone time

    Click Connect BuildPlanner folder (top-right) and pick this folder:

    Documents\My Games\Path of Exile 2\BuildPlanner

    Chrome, Edge, Brave & Opera save into it directly. Firefox/Safari download the file instead — just drop it into that folder yourself.

  2. 2

    Decant a build

    Press Decant on the headline build or any row in the ledger below. Tincture writes a .build file straight into your BuildPlanner folder.

  3. 3

    Open it in-game

    Launch Path of Exile 2 and open the Build Planner — your decanted build is waiting there, ready to follow node by node as you level.

No account, no login, nothing tracked — Tincture only ever writes to the one folder you choose, right in your browser. Don't see a BuildPlanner folder yet? Path of Exile 2 creates it automatically the first time you open the in-game Build Planner.
